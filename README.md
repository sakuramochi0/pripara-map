pripara-map
===========

Codes for Pripara Maps (http://sakuramochi.mydns.jp/pripara/map).

## Code structure

### Main programs

* csvmaker.py - fetch data from the official site to make it into csv in the csv directory. 
* make_html.py - combine csv and template to *.md and *-header.
* make_html.sh - convert *.md to *.html by using Pandoc.

### Other files

* google-map-link.csv - includes urls of Google Maps manually plotted places in the csv files.
* index-template, pref-header-template, pref-template - are templates for html.

## How to edit html files

When you edit text in the html files, you should change only *-template files. Because *.md (except README.md) and *-header which are automatically generated by make_html.py can be deleted.
